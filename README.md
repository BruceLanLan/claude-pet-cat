# 🐱 虚拟宠物宇宙 - Pixel Pet Universe

一个超级可爱、参考数码暴龙风格的像素化虚拟宠物养成游戏！

## ✨ 特性

- 🐾 **20种宠物** - 像素风格，每只都有独特外观和DNA
- 🧬 **DNA系统** - 宠物拥有HP/攻击/防御/速度/幸运属性
- ✨ **进化系统** - 达到10级可进化为更强形态
- 🎮 **互动系统** - 喂食/玩耍/抚摸/对话/治疗
- 🏆 **成就系统** - 收集成就解锁新宠物
- 🎁 **物品系统** - 食物/玩具/服装/药品
- 🧠 **思想系统** - 宠物有自己的想法和心情
- 📱 **双模式** - 自动模式 + 交互菜单

## 🚀 快速开始

```bash
# 克隆项目
git clone https://github.com/BruceLanLan/claude-pet-cat.git
cd claude-pet-cat

# 运行宠物宇宙
python3 my_pet.py --auto
```

## 🎮 使用方法

```bash
# 自动模式（后台运行，观察宠物动态）
python3 my_pet.py --auto

# 交互菜单模式
python3 my_pet.py -m

# 添加新宠物
python3 my_pet.py --add-pet spark
```

## 📋 状态说明

| 属性 | 说明 |
|------|------|
| 🍖 饱食度 | 0-100，低于20会饿 |
| 💖 快乐度 | 0-100，影响心情 |
| ⚡ 能量值 | 0-100，影响玩耍 |
| ❤️ 健康值 | 0-100，影响状态 |
| 💕 爱意 | 对主人的爱意值 |
