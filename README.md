# 🐱 Claude Pet Cat - 虚拟宠物猫

一个超级可爱、有思想、会自主行动的虚拟宠物养成游戏！

## ✨ 特性

- 🧠 **思想系统** - 猫咪有自己的想法，会根据心情变化
- 💕 **情感系统** - 14种情感：开心、兴奋、爱意、满足、顽皮、好奇、平静、无聊、困倦、饥饿、伤心、生气、害羞、寂寞
- 🧬 **性格系统** - 5种性格：甜心、调皮、安静、傲娇、粘人精
- ❤️ **爱意系统** - 对主人的爱意会随互动增长
- 🐾 **记忆系统** - 记住最近发生的事情
- 🎨 **彩色UI** - 可爱的彩色终端界面
- 🎬 **多种姿势** - 10+种可爱的ASCII猫咪姿势

## 🚀 快速开始

```bash
# 克隆项目
git clone https://github.com/anthropics/claude-pet-cat.git
cd claude-pet-cat

# 运行
python3 my_cat.py --auto --interval 4
```

## 🎮 使用方法

```bash
# 自动模式（推荐）
python3 my_cat.py --auto --interval 4

# 交互模式
python3 my_cat.py

# 改名
python3 my_cat.py --name 你的名字
```

### 交互模式操作
- 1. 查看状态
- 2. 喂食 🍗
- 3. 抚摸 💕
- 4. 玩耍 🎾
- 5. 抱抱 🤗
- 6. 说话 💬
- 7. 改名
- 8. 自动模式
- 9. 退出

## 📋 状态说明

| 属性 | 说明 |
|------|------|
| 🍖 饱食度 | 0-100，低于20会饿 |
| 💖 快乐度 | 0-100，影响心情 |
| ⚡ 能量值 | 0-100，低于20会困 |
| ♥ 爱意 | 对主人的爱意值 |
